# CV — Alexandr Lebedev

Адаптивное резюме на чистом HTML/CSS: семантическая разметка, аккуратная типографика и сетка на Flexbox/Grid.

## Демо
https://ruskapo.github.io/cv-alexandr-lebedev/

## Технологии
- HTML5 (semantic)
- CSS3 (Flexbox / Grid / адаптив)
- Mobile-first верстка

## Что внутри
- Разделы: Header, Summary, Skills, Projects, Experience, Education, Contacts
- Адаптив: mobile → tablet → desktop
- Печать: отдельные стили `@media print` (если добавишь — удобно для PDF)


## Как запустить локально
Открой `index.html` в браузере  
или подними простой сервер:

# VS Code: Live Server
# либо Node:
npx serve .
